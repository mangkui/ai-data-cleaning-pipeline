# AI-Powered Data Cleansing Pipeline

## Project Overview
This project demonstrates an AI-augmented data engineering workflow designed to automate the process of cleaning raw corporate data and instantly generating executive-ready KPI dashboards. 

By leveraging **Google AI Studio** and precise system instructions, this pipeline reduces manual data preparation time by roughly 80%, allowing analysts to focus entirely on data validation and strategic insight.

## System Workflow & Prompt Design
The core of this pipeline is anchored by structured prompt engineering. The underlying LLM was configured with the following system instructions:
1. **Schema Optimization:** Automatically identify and remove special characters or parentheses from messy column headers (e.g., converting `Price (Rs.)` to `Price_Rs`).
2. **Standardization:** Parse inconsistent date strings and convert them to SQL-ready ISO 8601 format (`YYYY-MM-DD`).
3. **Mathematical Precision:** Calculate and preserve floating-point accuracy for promotional discounts and final prices without losing row counts.

## Repository Structure
* `ecommerce_dataset_updated.csv`: The raw, unformatted sample dataset containing special characters and inconsistent string formatting.
* `cleaned_dataset_2026-06-26.csv`: The final, optimized dataset generated by the AI Studio orchestration layer.

## Operational Fallback Strategy
During development, the pipeline successfully managed a real-world server timeout (API 503 error) by dynamically shifting the backend orchestration layer to a high-speed variant (Gemini 1.5 Flash) and optimizing data packet sizes to guarantee business continuity.
